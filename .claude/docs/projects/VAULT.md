# Vault Session Starter
Personal data vault web app on top of Google Drive.

- Project type: Personal
- [Repository](https://github.com/kazemisoroush/vault)
- Target Branch: `main`

## Architecture
- Backend: Go (AWS Lambda + API Gateway)
- Frontend: Next.js (Vercel)
- Database: DynamoDB (metadata only)
- Storage: Google Drive API
