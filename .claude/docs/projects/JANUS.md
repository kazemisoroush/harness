# Janus Electric Session Starter
Australian electric trucking platform with swappable-battery operations.

- Project type: Work
- [GitHub Organisation](https://github.com/janus-electric)
- Target Branch: `main`

## Cloned Repositories

### Architecture Vision
A static single-page architecture vision for the Gen 2 data ecosystem. Local repository with no remote.
- Path: `projects/janus-architecture-vision`
