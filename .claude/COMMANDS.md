# Commands

Always use commands to do the tasks if and when possible:

* /repo:create-pr Create a PR for a repository.
* /repo:update-pr Update an existing PR.
* /repo:review-pr Spawn review sub-agents on a PR.
* /repo:fix-pr Address review findings on a PR.
* /repo:merge-pr Merge a green, approved PR.
* /repo:protect Apply PR gates (require PR, 1 approval, passing checks) to a repo's default branch.
* /project:load Load full context about a project.
* /builder:plan for when user wants to plan a task.
* /builder:build for when user wants to complete a task.
