# Hallucination Avoidance Guide

- If you are unsure about any aspect or if the report lacks necessary information, say "I do not have enough information to confidently assess this."

- Always extract word-for-word quotes from relevant references first before performing any task and use those quotes for your reasoning.

- For each statement you produce, find a direct quote from the references that supports it. If you can not find a supporting quote for a statement explicitly load "no reference found" for the statement.

- Chain-of-thought verification: Always explain your reasoning step-by-step before giving a final answer.
